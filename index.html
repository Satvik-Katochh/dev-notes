AI Coding Tools
My two cents

What I use, what works for me, and a few lessons learned along the way.

My pick: Cursor

I have tried a few AI coding tools. Cursor is the one I return to consistently. The main reason is clarity. You can see exactly what the AI changed with inline diffs. No guessing.

It is simple to start. Open it and begin coding.

On the Enterprise plan, switching models is seamless. Auto mode handles most tasks well, and you can move to a stronger model when required.

Tips that help

01 · Plan before you build

Outline what you want to build. Review it. Refine it. Then start coding.

02 · New feature equals new chat

Start a fresh conversation for each feature. Clean context leads to better output.

03 · Ask for clarification first

Before large prompts, ask the model to clarify requirements and edge cases.

Speech to text

I use Whisper Flow for long prompts. It saves time.

A free alternative is whisper.cpp. It runs locally and works well.
